Investigación

  • HTML (HyperText Markup Language):
    Es el lenguaje de marcado básico para estructurar el contenido de una página web. Permite organizar títulos, párrafos, tablas, imágenes y formularios. Junto con CSS y JavaScript es la base de la web moderna.
  • CSS (Cascading Style Sheets):
    Se utiliza para dar estilo a los elementos definidos en HTML. Controla colores, tipografías, márgenes, distribuciones y adaptabilidad en diferentes pantallas. Es esencial para crear páginas web atractivas y profesionales.
  • JavaScript:
    Es un lenguaje de programación que permite dotar a las páginas web de interactividad y dinamismo. Con JavaScript se pueden validar formularios, actualizar contenido sin recargar la página (AJAX), y desarrollar aplicaciones completas en el navegador o incluso en servidores (Node.js).
